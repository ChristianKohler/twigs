For a better user experience, we want to be able to react to mouseclicks anywhere on a table row, not just one link in a cell. This directive adds a mouse listener to the row and switches to the specified url when the user clicks anywhere on the row.

If the click originates from a anchor or element with ng-click attribute set, the click event that bubbles up will be ignored.
