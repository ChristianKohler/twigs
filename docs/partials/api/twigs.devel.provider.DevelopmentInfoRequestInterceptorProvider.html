DevelopmentInfoRequestInterceptorProvider
provider in module twigs.devel

Description

Use this provider to configure which XHTTPRequest to include in the development info footer.

var App = angular.module('Main',['twigs.devel']);

App.config(function (DevelopmentInfoRequestInterceptorProvider) {

// define your routes as usual
DevelopmentInfoRequestInterceptorProvider.setUrlFilter(/\/ws\/rest\//);

});

Methods

  • setUrlFilter(pattern)

    Pass in a regex pattern (e.g. /\/ws\/rest\//) to filter XHTTP request urls. Most of the time, you only want to include REST request and exclude angular's request to templates etc.

    Parameters
    ParamTypeDetails
    patternstring

    A regex pattern to filter XHTTP Request urls
