ExpressionEvaluator
service in module twigs.security

Description

ExpressionEvaluator is used by twigs.security directives. It validates and evaluates given expressions against the currently loaded user and its permissions.

Methods

  • evaluate(expression)

    Parameters
    ParamTypeDetails
    expressionstring

    The expression to evaluate This must match one of the following regular expressions:

    • /hasPermission(.*)/
    • /hasRole(.*)/
    • /isAuthenticated()/
    Returns
    boolean

    The return value of the evaluated expression.
