twgSecureEnabled
directive in module twigs.security

Description

Enables a input field only if the given expression evaluates to true. Allowed expressions are:

  • hasRole()
  • isAuthenticated()
  • hasPermission('some','arguments')

Usage

as attribute
<input twg-secure-enabled>
   ...
</input>

Example

<input type="text" twg-secure-enabled="hasPermission('entity','DELETE')"></input>
<input type="text" twg-secure-enabled="isAuthenticated()"></input>
