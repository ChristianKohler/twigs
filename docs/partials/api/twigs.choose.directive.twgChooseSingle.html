twgChooseSingle
directive in module twigs.choose

Description

Creates a ui-select2 choose dropdown element.

Attributes:

-model -> model variable which should be changed by selecting a value in the dropdown -choices -> array of choices to be displayed in the dropdown (one object in the choices array needs to be equal to the object in model for the mapping between model and dropdown to work properly) -choiceDisplayname -> name of the attribute of a choice to be displayed as text in the choose element

Usage

as element:
<twg-choose-single>
</twg-choose-single>
