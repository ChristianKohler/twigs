twgSecureShow
directive in module twigs.security

Description

Shows an element only if given expression evaluates to true. Allowed expressions are:

  • hasRole()
  • isAuthenticated()
  • hasPermission('some','arguments')

Usage

as attribute
<ANY twg-secure-show>
   ...
</ANY>

Example

<div twg-secure-show="hasPermission('entity','CREATE')">
</div>
